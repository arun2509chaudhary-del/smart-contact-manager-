Developed a Smart Contact Manager application to efficiently manage and organize user contacts with secure authentication and role-based access. Built using Spring Boot, Java, MySQL, JPA/Hibernate, and REST APIs, the system supports CRUD operations for adding, updating, deleting, and searching contacts. Implemented user login and registration with password encryption for enhanced security. Designed a responsive UI using Thymeleaf/HTML, CSS, and Bootstrap to ensure smooth user experience. The project demonstrates strong understanding of backend development, database management, and MVC architecture.
